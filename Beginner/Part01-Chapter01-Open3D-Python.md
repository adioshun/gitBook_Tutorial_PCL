## 2. 생성 

```python 
import open3d
import numpy as np

pc_array = np.array([[1, 2, 3], [3, 4, 5]], dtype=np.float32)
print(pc_array)

pc = open3d.PointCloud()
pc.points = open3d.Vector3dVector(pc_array)
print(pc)
```


## 3. 쓰기 

지원 하는 확장자 :  pcd, ply, xyz, xyzrgb, xyzn, pts

```python 
import open3d

open3d.write_point_cloud("pc2pcd.pcd", pc)
```
